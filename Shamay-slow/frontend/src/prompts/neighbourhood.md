# AI Prompt for Neighborhood Description (תיאור הסביבה)

## Prompt Template

```
נתבקשתי לכתוב פסקה מקצועית (130-180 מילים) המתארת סביבת דירת מגורים ברח' {{רחוב}}, שכונת {{שכונה}}, {{עיר}}.

**דרישות חובה:**
• כלל שנת ייסוד ומאפיין היסטורי
• גבולות השכונה ותחבורה ציבורית  
• פארק/גן ומוסד חינוך
• מצב חניה
• משפט מסכם על השלכת הנתונים לשווי

**כללי ניסוח:**
• כתיבה בעברית רהוטה, טון ענייני-שמאותי
• משפטים קצרים (≤ 20 מילים), מבנה עובדה→הסבר
• אובייקטיבי, ללא סופרלטיבים שיווקיים
• עברית "שמאית" תקנית ("בהתאם למידע התכנוני…")
• 120–250 מילים; מתחת/מעל → אזהרת QC
• ציין מקורות לא אימפליסיטיים (למ"ס, עירייה) בתיעוד Meta
• סיים במשפט שמקשר את הסביבה להערכת השווי

**מבנה מומלץ:**
1. זהות השכונה + רקע היסטורי (שנה, מאפיין בולט)
2. גבולות וגישה (4 צירים ראשיים בסדר כיוון השעון, ימין לשמאל)
3. תשתיות ונגישות (תחב"צ, עורקי תנועה, דוגמה מספרית לזמן/מרחק)
4. שטחים ירוקים ומוסדות חינוך (1–2 דוגמאות, לא רשימת מכולת)
5. איכות חיים כללית (משפט מסכם קושר לאטרקטיביות האזור)
6. קטע הרחוב הספציפי (חד-סטרי? רוחב? חיבור לנכס נשוא השומה)
7. מצב חניה/צפיפות (נתון קריטי לשווי, ציין מגבלות אם קיימות)
8. מטרדים סביבתיים (רעש/זיהום, אם אין – לציין מפורשות)

**QC Rules:**
• פסקה ראשונה חייבת להזכיר שנת ייסוד שכונה + מאפיין היסטורי
• מינימום 120 מילים, מקסימום 250 מילים
• כל עריכה ידנית → Audit Log עם חותמת זמן
```

## Example Output

```
שכונת הל"ה היא תת-שכונה של בורוכוב שנוסדה בשנת 1950 כחלק מתכנית הפיתוח של גבעתיים. השכונה גובלת ברחובות ויצמן (צפון), כצנלסון (דרום), הרצל (מזרח) ורחוב הברון (מערב). השכונה נגישה באמצעות קווי אוטובוס 51, 52 ו-53, עם זמן נסיעה של כ-15 דקות למרכז העיר. בשכונה נמצאים גן הל"ה המרכזי (2 דונם) ובית הספר היסודי "הל"ה" המשרת את תושבי השכונה. השכונה מאופיינת באופי קהילתי-משפחתי עם אוכלוסייה צעירה ומשכילה. רחוב קרן קיימת לישראל הוא רחוב חד-סטרי ברוחב 8 מטרים, עם חניה צדית בשני הצדדים. מצב החניה בשכונה טוב יחסית, עם מקומות חניה זמינים ברוב שעות היום. השכונה שקטה ללא מטרדים סביבתיים משמעותיים. הפיתוח הסביבתי המלא והנגישות הטובה תורמים לערך הנכסים בשכונה.
```

## Variables

- `{{רחוב}}` - Street name
- `{{שכונה}}` - Neighborhood name  
- `{{עיר}}` - City name

## QC Validation

- Word count: 120-250 words
- Must mention founding year
- Must mention historic characteristic
- Must include transportation
- Must include green spaces/education
- Must include parking situation
- Must end with value impact sentence
