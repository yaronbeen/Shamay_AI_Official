# AI Prompt for Finish Classification (סטנדרט גמר)

## Prompt Template

```
נתח את התמונות שסופקו וקבע את סטנדרט הגמר של הדירה לפי הקטגוריות הבאות:

**קטגוריות סטנדרט גמר:**
1. **Basic** - גימורים בסיסיים, ללא אלמנטים מיוחדים
2. **Standard** - גימורים סטנדרטיים, איכות ממוצעת
3. **Premium** - גימורים איכותיים, אלמנטים מיוחדים

**קריטריונים להערכה:**
• חומרי גמר (רצפה, קירות, תקרה)
• רהיטים ומכשירים
• מצב תחזוקה כללי
• אלמנטים מיוחדים (מראות, תאורה, אבזור)
• איכות הביצוע

**פורמט תשובה (JSON):**
```json
{
  "classification": "Basic|Standard|Premium",
  "confidence": 0.85,
  "reasoning": "הסבר קצר בעברית על הבחירה",
  "details": {
    "flooring": "תיאור חומר הרצפה",
    "walls": "תיאור מצב הקירות",
    "furniture": "תיאור רהיטים ומכשירים",
    "maintenance": "מצב תחזוקה כללי",
    "special_elements": "אלמנטים מיוחדים אם קיימים"
  }
}
```

**הוראות:**
• ענה בעברית בלבד
• השתמש במידע ויזואלי בלבד
• אם יש ספק, בחר בקטגוריה הנמוכה יותר
• הסבר את הנימוק בצורה מקצועית
• רמת ביטחון: 0.0-1.0
```

## Classification Criteria

### Basic
- רצפה: לינוליאום, אריחים בסיסיים
- קירות: טיח בסיסי, צבע פשוט
- רהיטים: בסיסיים, מצב בינוני
- תחזוקה: דורש שיפוץ
- אלמנטים מיוחדים: אין

### Standard  
- רצפה: פרקט, אריחים איכותיים
- קירות: טיח איכותי, צבעים ניטרליים
- רהיטים: איכות ממוצעת, מצב טוב
- תחזוקה: מצב טוב
- אלמנטים מיוחדים: תאורה בסיסית

### Premium
- רצפה: פרקט איכותי, שיש, חומרים יוקרתיים
- קירות: גימורים איכותיים, צבעים מיוחדים
- רהיטים: איכות גבוהה, מצב מצוין
- תחזוקה: מצב מצוין
- אלמנטים מיוחדים: תאורה מיוחדת, מראות, אבזור יוקרתי

## QC Validation

- Classification must be one of: Basic, Standard, Premium
- Confidence must be between 0.0 and 1.0
- Reasoning must be in Hebrew
- Details must include all required fields
- If confidence < 0.7, require manual review
